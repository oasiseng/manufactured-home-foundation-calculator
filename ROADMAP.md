# Product Roadmap

This roadmap focuses on making the calculator more useful for homeowners, installers, and engineers.

## Phase 1 (Implemented)
- Editable cost assumptions in the UI (block, pier, anchor, inspection costs).
- Shareable links that persist all current inputs and cost assumptions.
- Expanded engineering assumptions output table for better transparency.

## Phase 2 (Next)
- Side-by-side scenario compare mode (e.g., block vs pier).
- Export to PDF summary for permit/installer sharing.
- Basic location helper (ZIP code) with guidance for wind/seismic defaults.

## Phase 3 (Later)
- Add optional snow/frost/site slope inputs.
- Add labor, tax, and contingency cost model.
- Add user presets and saved projects.

## Success Metrics
- Increase share-link usage and return visits.
- Reduce support questions about "how totals were calculated."
- Improve conversion from estimate to professional inspection booking.
